02 — Formal Classification
01
By AI Architecture
Hybrid RAG Agent
Currently: Deterministic RAG — intent routing via curated keyword classifier against a hand-crafted knowledge store. Upgrade path: Hybrid RAG — playbook passed as LLM system prompt for probabilistic coverage with deterministic grounding. For compliance-critical outputs, deterministic retrieval is the correct choice.
02
By Agent Capability Class
Proactive Agentic AI (L3–L4)
Spans L1 (reactive Q&A) through L4 (governance + blocking). Most commercial AI assistants stop at L1–L2. Eggy's agentic field observer and compliance engine operate at L3–L4 — the definition of a "full agent" vs a "copilot."
03
By Enterprise Software Category
IPA + Compliance Automation
Conversational AI (chat layer) + Intelligent Process Automation (field observer + governance) + RegTech Compliance Agent (sanctions screening). Three recognized enterprise AI categories in a single embedded widget.
04
By Salesforce Taxonomy
Agentforce-Equivalent Agent
Maps precisely to: Agentforce Custom Topic (Q&A), Agentforce Custom Action/Flow (sanctions), Einstein Trust Layer analog (submission blocking), Prompt Builder (knowledge layer), Experience Cloud surface (PRM portal embed).
